### Global Club State Controller (GCSC)
- **inactive > surveying**
- [ ] 이제 sig와 pig를 생성할 수 있습니다.
- [ ] sig pig에 join을 할 수 있습니다

- **surveying > recruiting**
- [ ] 변경 시점에서 모든 신청인원은 가신청인원으로 전환됩니다.
- [ ] 여전히 sig와 pig를 생성할 수 있습니다
- [ ] sig pig에 join을 할 수 있습니다

- **surveying > recruiting**
- [ ] 변경 시점에서 모든 신청인원은 가신청인원으로 전환됩니다.
- [ ] 여전히 sig와 pig를 생성할 수 있습니다
- [ ] sig pig에 join을 할 수 있습니다

- **recruiting > active**
- [ ] 변경 시점에서, 봇이 가동됩니다. 단톡방을 만듭니다  
- [ ] sig과 pig 참가 신청은 할 수 없고, 목록을 볼 수만 있습니다 <- 기간 끝나도 참가 가능하게 할까?  
  
- **active > inactive**
- [ ] active인 일반 회원의 상태가 전부 pending으로 변경됩니다.
- [ ] 학기를 다음 학기로 바꿉니다
- [ ] sig와 pig 페이지가 내려갑니다 (근데 이건 학기 구현을 잘 만들면 자연스럽게 될 듯)
