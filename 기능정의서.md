## Commons

### 회원등록 흐름
- 이름입력
- 학번 입력 -> 이 시점에서 해당 정보로 유저 있는지 확인
- active한 유저 있으면 홈페이지로 이동
- pending 중인 유저 있으면 확인 후 /enroll로 등록
- 유저 없으면 전화전호 전공 입력받은 후 /create로 등록
